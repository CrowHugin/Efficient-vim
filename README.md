# Efficient Nvim

*requirement :* 
- *Nvim >= 0.10*
- Git >= 2.19.0
- Nerd Front **(optional)**


*installation:*

*If you already have an nvim configuration use:*

	mv ~/.config/nvim ~/.config/nvim.bak

Then:

    git clone https://github.com/CrowHugin/nvim_config ~/.config/nvim
